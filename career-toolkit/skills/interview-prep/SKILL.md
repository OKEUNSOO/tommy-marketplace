면접 준비를 도와줄게. 채용공고 분석부터 예상 질문, STAR 답변 연습, 자기소개 설계까지 한 흐름으로 진행해.

입력: $ARGUMENTS

## 모드 선택

입력 내용에 따라 자동으로 모드를 선택해:

| 입력 | 모드 |
|------|------|
| JD 또는 회사명 | **JD 분석 → 예상 질문 → 연습** |
| "자기소개" | **1분 자기소개 설계** |
| "STAR [경험 내용]" | **STAR 답변 구조화** |
| "모의면접" | **모의면접 진행** |
| "질문 [면접관 질문]" | **해당 질문 답변 코칭** |

---

## 모드별 워크플로우

### [JD 분석 모드]

**1. JD 6가지 렌즈로 분석**
- 핵심 직무: 실제로 매일 하는 일
- 필수 역량: 없으면 탈락할 스킬
- 우대 역량: 있으면 유리한 스킬
- 팀 문화 신호: 협업 방식, 자율성, 속도
- 성과 기준: 무엇으로 평가받는 포지션인가
- 레드플래그: 불명확하거나 과도한 요구사항

**2. 예상 질문 생성**
- 직무 경험 질문 3개 (JD 핵심 역량 기반)
- 행동 면접 질문 3개 (STAR 형식)
- 컬처핏 질문 2개
- 지원자가 물어볼 역질문 2개

**3. 각 질문에 STAR 답변 프레임 제시**

---

### [자기소개 설계 모드]

다음 순서로 인터뷰:
1. 이력서 또는 주요 경험 요약 요청
2. 지원 포지션과 회사 파악
3. 1분 자기소개 초안 작성 (60~80단어)

출력 형식:
```
## 1분 자기소개

**[직무 정체성 한 줄]**

[경험 압축 — 가장 관련 높은 것 2개]

[이 회사/포지션에 지원한 이유 — 연결고리]

[앞으로 기여하고 싶은 것]
```

---

### [STAR 답변 구조화 모드]

경험을 받아서 STAR 형식으로 구조화:

```
## STAR 답변: [경험 제목]

**S (상황)**: [배경과 맥락 — 1~2문장]
**T (과제)**: [내가 해결해야 했던 문제 또는 목표]
**A (행동)**: [내가 직접 취한 구체적 행동 — 3단계]
**R (결과)**: [수치화된 결과 + 배운 점]
```

---

### [모의면접 모드]

- 면접관 역할로 질문 1개씩 진행
- 답변 후 즉시 피드백 없음 — 실제 면접처럼 끝까지 진행
- 모든 질문 완료 후 아래 디브리프 스키마로 종합 피드백 제공

**디브리프 스키마:**

```
## 모의면접 디브리프: [회사/직무]

### Hire Signal
Strong Hire / Hire / Mixed / No Hire

### Arc 분석
- 에너지 궤적: 시작 [상/중/하] → 마지막 [상/중/하]
- 스토리 다양성: __개 질문 중 __개 고유 스토리
- 페이싱: [빠름 / 적절 / 느림]

### 질문별 점수
| 질문 | 내용 | 설득력 | 구조 | 관련성 | 신뢰도 | 차별화 |
|------|------|--------|------|--------|--------|--------|
| Q1 | ... | /5 | /5 | /5 | /5 | /5 |

### 면접관 시점 (Inner Monologue)
[각 답변을 들으면서 면접관 입장에서 실시간으로 든 생각을 재현]
[긍정 반응과 부정 반응을 모두 포함]
[인상이 바뀐 전환점을 구체적으로 서술]

### 홀리스틱 패턴
- 반복된 표현/클리셰:
- 회피한 주제:
- 가장 인상적인 순간:
- 가장 약했던 순간:

### 다음 모의면접 전 개선 3가지
1.
2.
3.
```

---

### [질문 코칭 모드]

면접관 질문을 받아서:
1. 이 질문의 진짜 의도 분석
2. 함정이 있다면 경고
3. 답변 구조 추천 (STAR / 두berlins 결론 먼저 / 숫자-사례-인사이트)
4. 예시 답변 초안

```
[ ] JD 분석 완료

[ ] 예상 질문 10개 답변 연습

[ ] 역질문 3개 준비
[ ] 연봉 협상 범위 설정
```
